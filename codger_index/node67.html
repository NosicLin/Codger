下标运算符()

语句形式：
expr ( <expr1> , ... , <exprn> )
其中n>=0, 转换成字节码为：
{
    expr
}
OP_ARRAY_BEGIN
{
    expr1
}
OP_ARRAY_PUSH
   ...
{
    exprn
}
OP_ARRAY_PUSH
OP_$(BRACKET)
其中:
1)当expr以成员访问结束时，例如circle.area()，BRACKET的值为CALL_WITH_HOST
2)其它的情况下，例如 cos(3)， BRACKET 的值为 CALL
