print语句

print语句用于将对象的值输出到标准输出端，语法规则用文法表示为：
print_stmt =  "print" [expr] { "," expr }
示例代码：
print 2                    #输出:  2
print "Codger"+"Is"+"Good" #输出:  "CodgerIsGood"
print 3+3,3+4,1+2          #输出:  6 7 3
a=[3,4,5]
print a                    #输出:  [3,4,5]
