Codger简介

Codger是一门面向对象，直译式的动态程序设计语言,Codger语言的目标是简化程序的编写。

下载与安装

git clone git://github.com/NosicLin/Codger.git 
cd Codger           #进入开载目录
git co master       #切换到主分支
make                #编译
sudo make install   #安装

运行Codger程序

codger <文件名>
