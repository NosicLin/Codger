Codger简介
=========
Codger是一门面向对象，直译式的动态程序设计语言,Codger语言的目标是简化程序的编写。

下载与安装
==================
```
git://github.com/NosicLin/Codger.git 
cd Codger 
make 
sudo make install
```

运行Codger程序
=========
```
codger <文件名>
```
